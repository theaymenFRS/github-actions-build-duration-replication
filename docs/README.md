# docs/

Ce répertoire contient la documentation nécessaire pour comprendre et reproduire les expériences.

- `replication_steps.md` — ordre recommandé des étapes d’exécution (commandes `py`, chemins `data/` et `results/`).
- `experimental_setup.md` — résumé de l’environnement logiciel et du protocole.
- `projects_description.md` — description des 10 projets sélectionnés et des fichiers de données associés.
- `troubleshooting.md` — problèmes fréquents lors de la réexécution des scripts archivés.

Voir aussi `README.md` à la racine du dépôt, `scripts/README.md` et `results/README.md` pour la structure du code et des sorties.
